Add the Ruttl bug tracking script to any part of your WordPress website.

== Description ==

The Ruttl Bug Tracking plugin allows you to integrate the Ruttl bug tracking system into your WordPress website with ease.

= Features =

* Easily configure Ruttl Bug Tracking on your WordPress site
* Exclude the Bug Tracker for logged-out users
* Optional inclusion or exclusion of using a filter

== Frequently Asked Questions ==

= How do I set the Project ID? =

Navigate to 'Settings' > 'Ruttl Bug Tracking Settings' and input your Project ID.

= How do I find my Project ID? =

Log in to your Ruttl account and navigate to the project you want to add to your WordPress website. The Project ID is the last part of the URL. For example, if the URL is \`https://web.ruttl.com/project/12345\`, the Project ID is \`12345\`.
